列表推导

涉及元素少,意味引入的缺陷少. 

    list_ = [ival for ival in range(10) if ival%2==0]
    print type(list_)
    print list_
    
    输出:
    <type 'list'>
    [0, 2, 4, 6, 8]

生成器表达式

    print type(list_)

    for el in iter:
