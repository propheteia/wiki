基本操作

创建数据库

使用管理员权限新建数据库,并给予账户权限

CREATE DATABASES test; GRANT ALL ON test.* TO 'user'@'localhost';

查看数据库

SHOW DATABASES;

选中数据库

USE test

查看表

SHOW TABLES;

创建表

-> CREATE TABLE pet (name VARCHAR(20), owner VARCHAR(20), -> species VARCHAR(20), sex CHAR(1), birth DATE, death DATE);

查看表的项目

DESCRIBE pet;

查看表的数据

SELECT * FROM pet;
