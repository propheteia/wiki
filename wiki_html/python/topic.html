安装PIL(Python Imaging Library) 模块

可能会报错

raise ImportError("The _imagingft C module is not installed")

那是因为缺少freetype库

"Your installed PIL was compiled without libfreetype"

在os上的解决办法:

brew install freetype pip install pil
